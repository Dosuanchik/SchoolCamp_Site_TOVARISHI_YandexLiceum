# Пришкольный лагерь Лицея №3 - Веб-сайт

Одностраничный сайт для пришкольного лагеря Лицея №3 г. Чебоксары


## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

3. Откройте в браузере:
```
http://127.0.0.1:8080
```
